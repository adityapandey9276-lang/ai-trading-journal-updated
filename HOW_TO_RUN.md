# ▶️ HOW TO RUN — THE MONEY PUZZELS

## ⚠️ IMPORTANT: Always run from app.py

```bash
cd trading_journal_updated
streamlit run app.py
```

Then open: http://localhost:8501

---

## ❌ DO NOT click the left sidebar links (admin, dashboard, etc.)
Those are Streamlit's auto-detected page links and will NOT work.
They have been disabled in this update by renaming the folder.

## ✅ USE ONLY the yellow buttons in the sidebar inside the app
